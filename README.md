# smart-retail-web

Retail store application build with Laravel

## Download Composer

`https://getcomposer.org/download/`

## Laravel

`composer global require "laravel/installer"`

## Github

`git clone https://github.com/frozen4code/smart-retail-web.git`

## Migrate

`php artisan migrate`

## Serve

`php artisan serve`
